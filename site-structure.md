# 个人学术主页内容结构

## 页面规划

### 1. Home (首页)
- 个人简介（简短版）
- 研究方向概述
- 最新动态/新闻
- 联系方式快速入口

### 2. Research (研究)
- 研究方向详细描述
- 主要研究项目
- 研究成果亮点
- 研究兴趣关键词

### 3. Projects (项目)
- 代码项目（GitHub 链接）
- 实验项目
- 工具/软件开发
- 项目描述、技术栈、成果

### 4. Publications (发表)
- 已发表论文
- 预印本
- 投稿中论文
- Working papers
- 引用格式（BibTeX）

### 5. Notes (笔记)
- 量子光学学习笔记
- QKD 技术笔记
- 安全性分析笔记
- 实验方法笔记
- 分类标签系统

### 6. CV (简历)
- 教育背景
- 研究经历
- 发表论文
- 技能清单
- 荣誉奖项
- PDF 下载链接

### 7. Contact (联系)
- 电子邮箱
- GitHub 主页
- Google Scholar 链接
- ORCID 链接
- 办公地址

## 技术实现方案

### 方案一：基于现有 Jekyll 模板（推荐）
- 优点：现有模板已适配，修改量小
- 缺点：Jekyll 构建较慢
- 实现：修改 `_config.yml`，添加新页面，更新导航

### 方案二：迁移到 Hugo
- 优点：构建速度快，主题丰富
- 缺点：需要迁移内容
- 实现：使用 Hugo Academic 主题

### 方案三：使用 Astro
- 优点：现代框架，性能好
- 缺点：学习曲线较陡
- 实现：使用 Astro 学术模板
