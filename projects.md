## Research Projects

### 1. High-Dimensional Time-Frequency Quantum Key Distribution

**Description**: Experimental and theoretical research on high-dimensional QKD systems based on SPDC sources, DWDM frequency multiplexing, and Franson interferometry.

**Key Research Areas**:
- DWDM-based frequency-bin encoding for high-dimensional quantum states
- Franson interferometry for time-frequency entanglement characterization
- Parameter Information Efficiency (PIE) and Secret Key Rate (SKR) analysis
- Time-frequency correlation characterization (JTI/JSI)

**My Contributions**:
- Experimental scheme design and implementation
- Time-frequency correlation data analysis
- PIE/SKR security performance evaluation
- JTI/JSI characterization methodology development
- Experimental workflow optimization

**Technologies**: SPDC photon sources, DWDM modules, Franson interferometers, TimeTagger, single-photon detectors

---

### 2. JTI Time-Domain Correlation Data Extraction Tool

**Description**: Development of an analysis pipeline for TimeTagger data processing and Joint Temporal Intensity (JTI) extraction, used to characterize the temporal correlations and high-dimensional mode structure of SPDC photon pairs.

**Key Features**:
- Coincidence pairing algorithms for photon pair identification
- Time-frame reconstruction for temporal correlation analysis
- Peak-aware matching for improved accuracy
- Schmidt number estimation for entanglement dimensionality
- Data visualization and statistical analysis

**My Contributions**:
- Implemented coincidence pairing algorithms
- Developed time-frame reconstruction methods
- Designed peak-aware matching strategies
- Performed Schmidt number estimation
- Created data visualization tools

**Technologies**: Python, NumPy, SciPy, Matplotlib, TimeTagger data formats

**Impact**: Improved the accuracy and reproducibility of temporal mode extraction for quantum source characterization.

## Experimental Projects

**Key Work**:

- Wavelength sweep measurements for source optimization

### 5. Quantum Interferometry Experiments

**Description**: Implementation of various quantum interferometry experiments for entanglement characterization.

**Experiments**:
- Hong-Ou-Mandel (HOM) interference for indistinguishability measurement
- Franson interference for time-bin entanglement
- Bidirectional frequency correlation (BFC) measurements
- Coincidence-to-accidental ratio (CAR) analysis
